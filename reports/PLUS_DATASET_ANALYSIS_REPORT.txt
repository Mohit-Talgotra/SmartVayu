STATISTICAL METHODOLOGY RECOMMENDATIONS
================================================================================
Based on the comprehensive statistical testing performed:

PRIMARY RECOMMENDATIONS:
1. USE NON-PARAMETRIC METHODS
   - Reason: Data fails all normality tests
   - Methods: Spearman correlation, Mann-Whitney U, Kruskal-Wallis
   - Avoid: T-tests, Pearson correlation assumptions, standard ANOVA

2. ACCOUNT FOR DEVICE EFFECTS
   - Reason: Significant inter-device differences detected
   - Methods: Mixed-effects models, device as random factor
   - Alternative: Analyze devices separately or apply calibration corrections

3. LEVERAGE NON-LINEAR RELATIONSHIPS
   - Reason: Spearman > Pearson correlations indicate non-linearity
   - Methods: Polynomial regression, spline fitting, machine learning
   - Focus: NH3-temperature relationship shows strongest non-linear pattern

4. ROBUST STATISTICAL APPROACHES
   - Reason: Heavy-tailed distribution with outliers
   - Methods: Robust regression, outlier-resistant methods
   - Validation: Bootstrap resampling for confidence intervals
